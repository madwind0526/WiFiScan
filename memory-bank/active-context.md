# Active Context

## Current Focus

- WifiScan은 현재 로컬 Wi-Fi/LAN의 휴대폰, PC, 가전, IoT 장비를 목록화하고 잠재적 보안 위험을 알리는 도구다.
- 공유기 정보, 로컬 호스트 탐색, 서비스 탐색을 결합해 장비 탐지 범위를 높인다.
- 단말 스캔만으로 모든 장비를 보장할 수 없으므로 관측 소스, 신뢰도, 누락 가능성을 표시한다.
- 기본 점검은 비침투 방식이며 비밀번호 대입, 취약점 악용, 무단 패치를 하지 않는다.
- 자동 대응은 공식 관리 API, 사용자 인증, 미리보기, 명시적 확인, 복구 경로가 있을 때만 허용한다.
- Android 17/API 37의 로컬 네트워크 권한을 포함해 버전별 권한 흐름을 분리한다.
- Wave 1의 도메인 모델과 네트워크 보안 대시보드 골격을 완료했다.
- Wave 2에서 Windows 활성 IPv4/게이트웨이/neighbor 탐색과 검색 진행·취소·목록 표시를 완료했다.
- Wave 3에서 MAC 우선 식별, 첫 검색 기준선, 로컬 JSON 스냅샷, 신규·변경·사라짐 비교를 완료했다.
- Wave 4에서 미확인 장비·신규 장비·검색 범위 제한을 근거 기반 경고로 연결했다.
- Wave 5에서 공식 커넥터가 없는 경우 수동 대응만 허용하는 안전 경계를 구현했다.
- Wave 6에서 Android 권한 안내, Kotlin 플랫폼 채널, 제한된 호스트 탐색 경로와 회귀 테스트를 구현했다.
- Android 실기기, mDNS/SSDP, 실제 공유기 제조사 API는 후속 확장 대상이다.
